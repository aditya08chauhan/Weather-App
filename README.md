# Weather Dashboard

A simple weather dashboard that provides **real-time, location-based weather information** in a clean interface.  
It fetches and displays weather data such as **temperature, conditions, and forecasts**, enhancing user convenience and travel planning.  

## Tech Stack
- **HTML, CSS, JavaScript**
- **OpenWeatherMap API**
- **Geolocation API**

## Features
- Detects user location using Geolocation API  
- Displays current temperature, weather conditions, and forecast  
- Responsive and user-friendly design
